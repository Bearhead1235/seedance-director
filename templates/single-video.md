# 单段视频分镜模板（≤15秒）

> **用途**：为单段视频（最长15秒）提供标准化分镜模板。根据视频类型选择对应模板，填入占位符即可生成完整分镜方案。
>
> **使用方法**：
> 1. 根据视频目标选择模板 A-E
> 2. 将 `[占位符]` 替换为具体内容
> 3. 根据分镜表生成即梦平台提示词
> 4. 参考「提示词转化」部分组装最终提示词

---

```

```

**即梦平台提示词占位框**：
```
[风格关键词] [画质关键词]。
画面从[起始景别]开始，[环境描述]，[角色外貌+动作]。
镜头[运镜方式]，[情节推进描述]。
随后[反转事件]，[角色反应]。
最终[收束画面]，[结尾意境]。
[色调]，[光影]，[摄影风格]。
```

### 使用说明

1. **起（0-3s）**：用大景别建立世界观，让观众快速进入情境。避免开头信息过载。
2. **承（3-7s）**：逐步拉近与角色的距离，通过景别变化传递情感升温。
3. **转（7-11s）**：这是全片情绪高点，运镜可适当激烈，配合音效制造冲击。
4. **合（11-13s）**：运镜回归平稳，给出情节结论。
5. **余韵（13-15s）**：留白是高级手法，不要塞太多信息，让画面自己说话。

### 提示词转化模板

```
[画面风格] [画质参数]，商业广告质感。
[产品名称]的极端特写开始，[材质/光泽描述]。

最终[品牌收尾画面]。

```

### 使用说明

1. **抓眼球（0-3s）**：前3秒决定留存率。用极端特写+快速运镜制造视觉冲击。
2. **全景展示（3-6s）**：让观众看清"这是什么"，环绕镜头最能展现产品立体感。
3. **细节特写（6-9s）**：这是卖货的核心段，用微距放大产品优势。
4. **场景使用（9-12s）**：让产品"活"起来，建立使用联想。
5. **品牌落版（12-15s）**：干净收尾，确保品牌记忆点。

## 模板E：短剧对白型

**适用场景**：短剧、情景喜剧、对话场景、采访片段、教学对话

### 提示词转化模板

```
[风格关键词] [画质参数]。
[场景描述]中，[角色A外貌描述]和[角色B外貌描述]面对面。
角色A说：「[台词]」，表情[情绪描述]。
角色B回应：「[台词]」，[动作/表情]。
[冲突/反转事件描述]，两人[反应描述]。
最终[结局画面]。
[色调]，[光影]，[摄影风格]。
```

### 使用说明

1. **人物亮相（0-2s）**：用中远景快速交代人物关系和空间布局。
2. **对话A（2-5s）**：过肩镜头是对白的经典机位，保持角色关联感。
3. **对话B（5-8s）**：与对话A形成正反打对称结构。
4. **冲突/反转（8-12s）**：打破对称、加入手持晃动，制造紧张感。
5. **结局（12-15s）**：后拉镜头天然具有"故事结束"的暗示。

> **即梦平台对白注意事项**：
> - 台词和画面描述分行书写，先写画面再写台词
> - 标注角色情绪词（愤怒、惊讶、温柔等）帮助AI理解表演状态
> - 音效单独描述，不要混在台词里
