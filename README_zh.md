# seedance-director

你的 AI 视频导演 — 从创意想法到专业分镜和即梦提示词，一步到位。

## 特性

- **影视方法论加持** — 融合三幕式、起承转合、Beat Sheet 等经典叙事框架，让 AI 视频有专业的故事结构
- **Seedance 2.0 全能力覆盖** — 支持纯文本生成、运镜复刻、特效复刻、一镜到底、视频延长等全部 10 大能力
- **自适应五阶段引导** — 新手有保姆级引导，老手可直接跳步，不浪费时间
- **单段 + 多段自动处理** — 15 秒以内单段搞定，超长视频自动拆段 + 衔接锚点，最长支持 60 秒以上
- **三层输出** — 专业分镜表 + 即梦提示词 + 平台操作指引，从创意到落地全覆盖
- **中英双语镜头术语** — 景别、运镜等专业术语同时标注中英文，学习和使用两不误
- **5 类场景模板** — 电商广告、仙侠特效、短剧叙事、科普教程、音乐 MV，开箱即用

## 快速开始

```bash
claude skill add --url https://github.com/crowscc/seedance-director
```

安装后，直接和 Claude 聊你的视频创意就行。提到「即梦」「视频提示词」「分镜」「AI 视频」「运镜」等关键词时，Skill 会自动激活。

## 工作流程

```
你的创意 → 理解创意 → 深度挖掘 → 生成分镜 → 生成提示词 → 操作指引
              Phase 1    Phase 2    Phase 3     Phase 4     Phase 5
```

**Phase 1 — 理解创意**：扫描你输入的 5 个核心维度（主题、时长、风格、素材、声音），判断信息完整度。

**Phase 2 — 深度挖掘**（自适应）：只追问缺失的维度，用选择题降低决策负担。比如推荐叙事结构（起承转合 / Hook-反转 / 对比型 / 悬念型等），你选一个就行。如果你的输入已经很详细，这一步直接跳过。

**Phase 3 — 生成分镜**：输出专业的逐镜头分镜表，包含时间戳、景别、运镜、画面描述、台词和音效。

**Phase 4 — 生成提示词**：把每个镜头转化为即梦 Seedance 2.0 可以直接使用的提示词，自动选择最合适的平台能力。

**Phase 5 — 操作指引**：告诉你在即梦平台上具体怎么操作 — 点哪个按钮、设什么参数、怎么把多段视频接起来。

## 使用示例

```
你：    我想做一个 30 秒的咖啡品牌广告，电影感，暖色调，
        我有 3 张产品图。

导演：  收到 — 电影风格咖啡广告，30秒，暖色调，有产品图。
        快速确认两个问题：

        叙事结构想用哪种？
        A. Hook → 揭示 — 开头蒸汽特写抓眼球，再展开全景
        B. 过程展示 — 从咖啡豆到杯中的旅程
        C. 生活方式 — 角色在温馨场景中享用咖啡

你：    B

导演：  [生成完整分镜]

        ## 分镜脚本：咖啡溯源

        **叙事结构**：过程展示（豆 → 研磨 → 品尝）
        **总时长**：30秒（2段）| **宽高比**：16:9 | **风格**：电影暖调

        | 镜号 | 时间  | 景别          | 运镜        | 画面描述                  |
        |------|-------|--------------|-------------|--------------------------|
        | 001  | 0-4s  | 微距 Macro    | 缓推 Dolly In | 晨光中的咖啡豆纹理        |
        | 002  | 4-8s  | 近景 Close-Up | 俯拍下摇     | 咖啡豆倒入研磨机          |
        | 003  | 8-12s | 中景 Medium   | 固定         | 手冲咖啡过程，蒸汽升腾     |
        | 004  | 12-15s| 特写 Close-Up | 缓推         | 咖啡液滴落，涟漪荡开       |

        ## 第 1 段提示词（0-15s）
        @产品图1 中的咖啡豆在晨光中缓缓滚动，微距镜头捕捉豆面油亮质感，
        缓推至近景。咖啡豆倒入铜制手摇磨豆机，俯拍下摇跟随豆子落下...
        电影质感，暖色调，浅景深，自然光线。

        ## 第 2 段提示词（延长 15s）
        将 @视频1 延长15秒。
        接续画面：热咖啡倒入白色陶瓷杯中，液面泛起细腻油脂花纹...

        ## 操作指引
        1. 打开即梦 → 选择「图生视频」
        2. 上传产品图 1，粘贴第 1 段提示词
        3. 时长设置 15 秒，宽高比 16:9
        4. 生成后，选择「视频延长」
        5. 上传第 1 段视频，粘贴第 2 段提示词
        6. 延长时长设置 15 秒
        7. 两段自动拼接，总时长 30 秒
```

## 支持的 Seedance 2.0 能力

| # | 能力 | 说明 |
|---|------|------|
| 1 | **纯文本生成** | 无需素材，纯靠文字描述生成视频 |
| 2 | **一致性控制** | 通过参考图锁定角色外观和场景风格 |
| 3 | **运镜复刻** | 借鉴参考视频的镜头运动方式 |
| 4 | **特效复刻** | 借鉴参考视频的视觉特效风格 |
| 5 | **剧情补全** | AI 自动补全关键帧之间的动画过渡 |
| 6 | **视频延长** | 在已有视频基础上续写新内容 |
| 7 | **声音控制** | 控制台词、音色和音效 |
| 8 | **一镜到底** | 多张图片串联成连续不切镜的长镜头 |
| 9 | **视频编辑** | 修改已有视频中的特定元素 |
| 10 | **音乐卡点** | 视觉节奏与音乐节拍精准同步 |

## 超长视频分段策略

| 总时长 | 段数 | 策略 |
|--------|------|------|
| ≤15s | 1 段 | 单次生成，时间戳分镜法 |
| 16-30s | 2 段 | 第 1 段正常生成 + 1 次视频延长 |
| 31-45s | 3 段 | 第 1 段正常 + 2 次延长 |
| 46-60s | 4 段 | 第 1 段正常 + 3 次延长 |
| >60s | 按场景拆分 | 独立场景分别生成 + 剪辑软件拼接 |

每段之间设置**衔接锚点** — 上一段的结尾画面 = 下一段的起始画面描述，确保视觉连贯。
