# seedance-director 项目文档

## 项目概述

创建一个高质量的 Claude Code Skill，结合传统影视方法论和即梦 Seedance 2.0 平台能力，通过交互式引导帮助用户生成专业的视频提示词。

## 核心定位

**AI 导演** — 不只是写提示词，而是从脚本、分镜、镜头语言全流程指导视频创作。

## 已确认的设计决策

| 决策项 | 选择 | 说明 |
|--------|------|------|
| 目标用户 | 新手+老手兼顾 | 新手有引导，老手可跳步 |
| 工作流 | 自适应流程 | 根据输入详细程度自动跳过已知步骤 |
| 影视方法论深度 | 分层融入 | 通俗语言引导 + 专业术语标注 |
| 超长视频策略 | 半自动 | 自动拆段，逐段确认 |
| 输出格式 | 分镜文档 + 提示词 + 操作指引 | 三层输出 |
| 命名 | seedance-director | 强调「导演」角色 |
| 仓库组织 | 单 Skill 仓库 | 仓库名即 Skill 名 |
